# CRM-project (Frontend)

Customer Relationship Management project for CSE327

## Frontend (Vite + React + Tailwind)

### Install
1. cd frontend  
2. npm install  
3. npm run dev  

### Notes
- Tailwind needs PostCSS; this scaffold includes basic config files.

### Added Pages
- Contacts  
- Deals  
- Tasks  

Use the navigation to reach the pages.
